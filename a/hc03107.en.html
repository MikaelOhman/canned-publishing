Abolished stamp duty for registering mortgages on ships.

Source Riksdagen Read the original at Riksdagen →2025-03-11
TL;DRThe government intends to abolish a tax that is currently levied when a ship is used as collateral for a loan. This will make it cheaper to register loans secured by ships, starting from July 1, 2025.
In plain language

Currently, there is a rule that requires individuals to pay a fee to the state, known as a stamp duty, when applying to use a vessel as collateral for a loan. This is called a maritime lien. This fee currently costs four Swedish kronor for every thousand kronor the loan is worth.

The government has now proposed eliminating this tax entirely. If the proposal is approved, individuals will no longer need to pay this fee to the state when registering the loan against the vessel. The purpose is to amend the laws governing stamp duty and maritime law, which are the laws that regulate this process.

If the parliament votes in favor of the proposal, the new rules will come into effect on July 1, 2025. This means that anyone who needs to use a vessel as collateral for a loan will no longer have to pay this additional cost after that date.

Full text

The proposal suggests abolishing the stamp duty currently levied when applications for ship mortgages are approved. According to current regulations, a stamp duty must be paid to the state when an application for a ship mortgage is granted. This stamp duty is currently four Swedish kronor for each full thousand kronor of the amount being mortgaged. The proposal would eliminate this stamp duty.

The proposed legislative changes are scheduled to come into effect on July 1, 2025.
