Requisitos para los sistemas de gestión de expedientes de las autoridades catastrales municipales

Fuente Riksdagen Leer el original en Riksdagen →2026-04-28
En resumenMunicipios que gestionan asuntos relacionados con la cartografía y el registro de la propiedad deberán utilizar, a partir de 2027, el mismo sistema digital que la agencia estatal Lantmäteriet. Esto se hace para garantizar que la tecnología funcione de manera compatible entre los municipios y el estado.
En lenguaje sencillo

El gobierno ha presentado una propuesta para modificar las normas de las oficinas municipales de catastro. Una oficina de catastro es la parte de un municipio que se encarga de asuntos relacionados con el terreno y los límites. A partir del 1 de enero de 2027, estas oficinas municipales estarán obligadas a utilizar el mismo sistema digital que la agencia estatal de catastro, Lantmäteriet, para gestionar sus expedientes.

El objetivo de este cambio es garantizar que las herramientas y los recursos técnicos para la gestión sean los mismos. Se trata de facilitar el intercambio de información de forma segura entre los municipios y el estado. Si un municipio no utiliza los sistemas adecuados, el gobierno puede revocar su autorización para operar una oficina de catastro.

El gobierno, o una agencia designada por el gobierno, también puede establecer las normas sobre cómo se deben utilizar estos sistemas técnicos. Además, pueden determinar si el uso de estos sistemas implica un costo a través de diversas tarifas.

La propuesta implica que los requisitos para que un municipio pueda establecer su propia oficina de catastro serán más claros. Esto incluye, entre otras cosas, contar con el personal adecuado, una organización correcta y que la tecnología funcione para el intercambio de información entre el estado y el municipio.

Texto completo

En la propuesta se sugieren modificaciones a la ley (1995:1393) sobre la autoridad municipal de catastro, que introducen un requisito de que las autoridades municipales de catastro utilicen el sistema de gestión de casos y otras herramientas técnicas de apoyo que proporciona la autoridad estatal de catastro.

Se propone que esta modificación entre en vigor el 1 de enero de 2027.
