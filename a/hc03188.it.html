Proposta di legge

Una detrazione di base nell'imposta sul rischio.

Fonte Riksdagen Leggi l'originale su Riksdagen →2025-06-24

- La proposta prevede l'introduzione di una deduzione base nella legge sulla tassa sui rischi per gli istituti di credito. La proposta stabilisce che un istituto di credito svedese o un istituto di credito estero debba effettuare una deduzione base dalla base imponibile per la tassa sui rischi. L'importo della deduzione base deve corrispondere alla soglia di imponibilità vigente per l'anno fiscale in questione. Se un istituto di credito fa parte di un gruppo con più istituti di credito, la deduzione base deve essere ripartita tra gli istituti di credito membri in base alla loro quota della base imponibile complessiva del gruppo all'inizio dell'anno.

Per non compromettere lo scopo della tassa sui rischi, la proposta deve essere finanziariamente neutra per le finanze pubbliche. Pertanto, la proposta prevede anche un aumento dell'aliquota della tassa sui rischi dallo 0,06 allo 0,07% della base imponibile all'inizio dell'anno.

Le nuove disposizioni dovrebbero entrare in vigore il 1° gennaio 2026.
