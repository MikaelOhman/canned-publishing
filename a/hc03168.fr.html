Impôts fonciers plus élevés pour les éoliennes.

Source Riksdagen Lire l'original sur Riksdagen →2025-06-17

- Proposition concernant l'augmentation de la taxe foncière sur les éoliennes, passant de 0,2 % à 0,5 % de la valeur imposable. La taxe foncière est due, entre autres, pour les biens classés comme unités de production d'électricité lors de l'évaluation foncière. Pour les unités de production d'électricité constituées d'une unité d'évaluation comprenant une éolienne, la taxe foncière est de 0,2 % de la valeur imposable. Pour les autres unités de production d'électricité, la taxe foncière est de 0,5 % de la valeur imposable. Cette proposition vise à uniformiser le taux de taxe pour toutes les unités de production d'électricité.

La modification législative proposée entrerait en vigueur le 1er janvier 2026.
